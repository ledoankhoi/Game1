# MathQuest - Game 1 🚀

**Full Documentation**: [docs/MathQuest-Full-Documentation.md](docs/MathQuest-Full-Documentation.md)

## Quick Start
```bash
# Backend (port 5000)
cd game-backend
npm install
npm run dev

# Frontend (port 5173) 
cd portal-ui
npm install
npm run dev
```

**Play**: http://localhost:5173

## Features
- 10+ Math/Logic Mini-games
- RPG Coins/EXP/Levels
- Shop & Avatar Skins (Gundam)
- Leaderboard & Profile
- Admin Panel
