# 📚 TÀI LIỆU HOÀN CHỈNH DỰ ÁN **MATHQUEST - GAME 1** (BLACKBOXAI)

## 🎯 **TỔNG QUAN DỰ ÁN**
**MathQuest** là nền tảng **game trí tuệ toán học/logic puzzle** kết hợp **RPG progression system**:
- **10+ Mini-games**: Chess logic, Maze, Puzzle, Sequence, Monster, v.v.
- **Economy**: Kiếm **coins/EXP** → lên level → mua **skins/items** (Gundam themes).
- **Social**: Leaderboard, Profile customization, Admin dashboard.
- **Multi-platform**: Web (React) + Backend (Node/Express/MongoDB).

**Repo**: https://github.com/ledoankhoi/Game1
**Trạng thái**: **Production-ready**.
